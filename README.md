# tsengine

Go-native two-layer security + compliance engine.

- **L1** — complete OSS vulnerability discovery for security + compliance teams
- **L2** — AI security and compliance engineer (developers / PMs audience; not built yet)

Paired with `webappsec` (the SaaS wrapper that consumes tsengine output).
